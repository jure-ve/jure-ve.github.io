Áreas de Interés Profesional

Desarrollo de Aplicaciones

Tengo una amplia experiencia como programador Delphi, c++, C# y Java desarrollando e implementando componentes y aplicaciones personalizadas para los diferentes modelos de negocio como lo son el comercio, puntos de venta, logística, comunicaciones y otras industrias. Me especializado en el diseño de base de datos relacionales y desarrollo de procedimientos en SQL en varios sistema de administración y gestión como SQL Server, MySQL/MariaDB y Firebird. Actualmente desarrollo aplicaciones Android. Tengo la habilidad para analizar los problemas y buscar las mejores soluciones para optimizar y gestionar procesos.

Desarrollo Web

Soy desarrollador senior con amplio conocimiento de las técnicas front-end y back-end. He participado en numerosos proyectos de diseño, desarrollo e implementación de aplicaciones para el entorno web. Soy hábil para programar con lenguajes como PHP, Python y Javascript, ademas del uso de varios frameworks entre los que puedo mencionar Codeigniter, Symfony, Slim y Laravel para PHP, por el lado de Python podría nombrar web2py, flash, bottle y Django. También tengo experiencia con Express.js en el entorno Node.js. Experto en integración de código y buena capacidad para la resolución de problemas durante el desarrollo web. Me interesa especialmente la usabilidad y los estándares web.

Administrador de Sistemas

Experiencia en la instalación, configuración, gestión, monitorización, respaldo y migración de sistemas basados en sistemas operativos Linux y Windows. Proyecto, implementación y puesta a punto de servidores y servicios web, base de datos, servidores de archivos, correo, redes, planes de seguridad y organización de directorios de usuarios y recursos. Análisis, simplificación y automatización de procesos mediante scrips en bash y powershell. Resguardo del correcto funcionamiento de servicios y planes de respaldo.
